# Mini Attendance + Task System

##  Project Overview
Ye project ek **Mini Attendance + Task Management System** hai jisme:

- Users signup/login kar sakte hain (JWT authentication)
- Attendance mark kar sakte hain (ek user ek din me sirf ek baar)
- Task create, update, aur list dekh sakte hain
- Frontend React + Tailwind CSS me banaya gaya hai
- Backend Node.js + Express + MongoDB hai
- Backend secure aur deployable cloud-ready hai
